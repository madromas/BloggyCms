# BloggyCMS

**Современная CMS с открытым исходным кодом для блогов и сайтов.**

[![PHP Version](https://img.shields.io/badge/PHP-8.0+-blue.svg)](https://php.net)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

BloggyCMS — это гибкая и расширяемая платформа, где каждый компонент можно адаптировать под свои нужды без изменения ядра. Всё построено на модульной архитектуре, системе событий и функциональном API.

## ✨ Особенности

### Архитектура

| Компонент | Описание |
|-----------|----------|
| **MVC-подобная структура** | Разделение на контроллеры, модели и представления |
| **Модульная система действий** | Actions для каждого контроллера |
| **Гибкие html-блоки** | Которые работают и как плагины, и как самостоятельные модули |
| **Событийная система** | Хуки и триггеры для кастомизации |

### Управление контентом

- **📄 Посты и страницы** — с категориями, тегами и архивами
- **🧩 Блочный редактор (PostBlocks)** — гибкое построение контента
- **🔧 HTML-блоки** — вставка готового кода
- **📦 Пользовательские поля (Custom Fields)** — для любых сущностей:

| Тип поля | Описание |
|----------|----------|
| `Текст`, `Текстовая область`, `Число`, `Дата`, `Уведомление` | Базовые типы |
| `Выпадающие списки`, `Мультивыбор`, `Чекбоксы` | Выбор из вариантов |
| `Изображения`, `Цвет`, `Ссылки`, `Иконки` | Медиа и ссылки |
| `Повторители (Repeater)` | Повторяющиеся группы полей |

### Пользователи и права

- **👥 Группы пользователей** — гибкая система прав доступа
- **👤 Профили пользователей** — любые кастомные поля
- **🏆 Система достижений (Achievements)** — автоматическая разблокировка
- **💬 Модерация комментариев** — права на редактирование/удаление

### Шаблоны и темы

- **🎨 Поддержка нескольких тем** оформления
- **📁 Кастомные шаблоны** — для блоков в папке темы
- **📦 Asset-менеджер** — подключение CSS/JS файлов

## 💻 Требования

| Компонент | Версия |
|-----------|--------|
| **PHP** | 8.0 + |
| **MySQL** | 5.7 или выше |
| **Веб-сервер** | Apache/Nginx с mod_rewrite |

---

### 🤷‍♂️ Почему PHP 8.0, а не 5.6?

Потому что 2026 год на дворе, а не 2012.

- **🚀 Скорость** — до 30% быстрее. Тормоза остались в прошлом
- **🛡️ Безопасность** — обновления, а не "ну, может, не взломают"
- **✨ Современный код** — match, named arguments, attributes.
